# Template: Client + Server

A full-stack app with an Express 5 backend and a Vue 3 frontend. Use this when you need API routes, secret management, or server-side logic alongside a client.

## Stack

Express 5 + Vue 3 + Vite + Pinia + tsx (dev hot-reload)

## Usage

```sh
pnpm dev        # starts server and Vite dev server in parallel
pnpm build      # compile both server and Vue client
pnpm start      # run production build
pnpm test       # run tests
```
